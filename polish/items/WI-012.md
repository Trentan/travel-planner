## Problem

`.modal-close` uses only `padding: 0.25rem` and no min-width/min-height. The visible close icon is large, but the guaranteed tap target is not defined.

## Proposed fix

Set `.modal-close` to `min-width: 44px; min-height: 44px; display: inline-flex; align-items: center; justify-content: center;`.


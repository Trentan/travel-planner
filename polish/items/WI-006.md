## Problem

The transport tab rendered as a dense card-like mobile layout. The table columns were not clearly separated, and the header did not stay visible while scrolling.

## Proposed fix

Keep the transport view as a table on mobile, but simplify it to four clear columns: Journey, Schedule, Status/Cost, and Actions. Add a sticky mobile header row so the column labels remain visible while scrolling.
